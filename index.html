HASTE

This was done for my 2nd project as part of the General Assembly coursework. We had just learnt about Node and Express and were told to build an app with them. And it happened that my friends, who run a small chair business, needed something to help organise their deliveries.

Features:

  • Index of orders/deliveries + summary bar
  • Integrated Google Maps
  • Integrated OneMap API for postal-to-coordinates conversion
  • Data stored in MongoDB cloud
  • Staff login using Passportjs

Sorting visualiser

Currently, the sorter runs 4 sorting algorithms: Quick, Merge, Bucket, and Heap. I hope to implement the final, king-of-all-sorts sort -- the Bogosort.

As for the demo, if you like seeing things change really quickly on the board, you may set the slider to the maximum. If you prefer seeing how exactly the algorithm is working, you may slide it to the minimum. And press sort, of course.
